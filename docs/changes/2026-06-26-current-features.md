# 个人 Markdown 云笔记现有功能说明

日期：2026-06-26

## 概览

这是一个面向个人局域网使用的 Markdown 云笔记网站。电脑作为唯一数据源和服务端，网页端与 Android APP 都连接电脑服务读写同一份 Markdown 文件和附件。

## 文档与目录管理

- 支持无限级目录树。
- 支持新建目录、新建文档、重命名、移动、拖拽移动和移到回收站。
- 文档内容保存为普通 `.md` 文件，目录结构对应硬盘上的文件夹结构。
- 支持回收站恢复、永久删除和清空。
- 支持最近文档视图，按文件更新时间查看近期笔记。
- 支持全量 ZIP 导出，包含笔记、附件和回收站内容。

## Markdown 编辑

- 使用 Milkdown/Crepe 提供类飞书的所见即所得 Markdown 编辑体验。
- 支持正文、H1-H5、引用、代码块、分割线、链接、图片、附件、GFM 表格、高亮块和任务列表等常用块。
- 支持 `/` 插入菜单，插入基础块、列表、图片、代码块和表格。
- 选中文字后显示浮动工具栏，支持加粗、斜体、删除线、行内代码、链接和格式切换。
- 格式切换支持把已选文字直接改成正文或 H1-H5。
- 中文斜体已做视觉兼容处理。
- 代码块支持粘贴后自动推断常见语言。

## 文档内目录

- 编辑器右侧显示当前文档的标题目录。
- 目录从 Markdown 中的 H1-H5 自动生成，不写入笔记文件。
- 有子标题的目录项支持展开和收起。
- 点击目录项可跳转定位到对应标题。
- 手机端通过顶部“文档目录”按钮打开抽屉式目录。

## 自动保存与冲突处理

- 编辑内容后自动保存，默认延迟约 800ms。
- 保存状态显示等待保存、保存中、已保存和保存失败重试。
- 自动保存不会在保存响应后重建编辑器，避免输入时光标丢失。
- 如果检测到 revision 冲突，不会静默覆盖服务器版本，会提示加载最新版本或另存为副本。

## 附件与图片

- 支持上传图片并在 Markdown 中内联显示。
- 支持上传普通附件并作为链接插入。
- 支持粘贴、拖入和文件选择上传。
- 只有安全图片类型允许内联预览，普通附件按下载方式访问。
- Android APP 访问电脑服务时，附件和图片 URL 会转换为可访问的局域网服务地址。

## 搜索

- 支持搜索标题与正文。
- 搜索索引在服务端内存中维护，可由 Markdown 文件重新构建。
- 支持中文分词式检索，适合搜索笔记关键词。

## 交互体验

- 纯图标按钮补充了中文 Tooltip 和无障碍名称。
- 按钮、菜单、对话框、抽屉和列表变化有短过渡动画。
- 支持 `prefers-reduced-motion`，系统偏好减少动态效果时会关闭动画。
- 桌面端为导航、目录树、编辑器三栏布局。
- 手机端使用目录抽屉、全宽编辑器和底部插入工具栏。

## Android APP

- 使用 Capacitor 打包 Android APP。
- APP 名称为“个人笔记”，包名为 `com.personal.markdownnotes`。
- APP 不保存 Markdown 副本，只作为局域网客户端连接电脑服务。
- 首次启动输入电脑服务地址，例如 `http://电脑IP:3210`。
- 支持保存最多 5 个历史服务器地址，并可测试、切换和删除。
- 支持 Android 返回键关闭菜单、对话框、目录抽屉或返回笔记视图。

## 部署与启动

- 默认服务端监听 `0.0.0.0:3210`，可通过 `.env` 配置 `HOST`、`PORT`、`DATA_DIR` 和上传限制。
- 支持 `npm run dev` 开发启动。
- 支持 `npm run build` 后用 `npm start` 启动生产服务。
- 提供 Windows 开机自启和防火墙放行脚本。
- 电脑服务启动后，局域网设备可通过电脑 IP 访问。

## 当前限制

- 面向可信局域网个人使用，没有登录、权限和 HTTPS。
- 不支持多人实时协作、评论、分享权限和版本历史。
- Android APP 不支持离线编辑和离线同步。
- 目前不内置定时备份，主要通过 ZIP 导出手动备份。
- 对齐和缩进功能暂未实现。
