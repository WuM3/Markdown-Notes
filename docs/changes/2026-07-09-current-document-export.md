# 2026-07-09 当前文档导出与插入菜单修复

## 插入菜单

- “插入”菜单改为 Radix 非模态模式，不再在打开时锁定整个页面交互层。
- 菜单动效移除缩放，只保留淡入和 4px 位移，避免含 CodeMirror 代码块时出现整页瞬时闪黑。
- 回归测试覆盖代码块文本、折叠状态、滚动位置和菜单关键帧。
- 修复格式菜单关闭后鼠标落到下层按钮、误开颜色菜单并遮挡编辑器的问题。

## 当前文档导出

- 删除左侧“导出全部笔记”入口和旧的全量 ZIP API。
- 在编辑器右上角增加“导出”，位于“插入”左侧。
- 支持 Markdown、Word 和 PDF，文件名支持中文并过滤 Windows 非法字符。
- 导出使用编辑器当前草稿，不等待自动保存，也不会修改服务器 revision。
- Markdown 保留 frontmatter；Word/PDF 保留标题、样式、列表、引用、代码块、表格、链接和本地图片。
- 损坏、缺失、外部或越界图片降级为替代文字，不中断其余内容。
- 手机端使用图标按钮，避免顶部工具栏溢出。

## 测试

- 导出模型覆盖空文档、Unicode、组合样式、嵌套列表、任务、引用、代码块、宽表格、资源和恶意 HTML。
- API 覆盖三种文件签名、中文 PDF、未保存草稿、参数校验、缺失文档、损坏图片、路径穿越和并发隔离。
- 客户端覆盖二进制错误、文件名解析、对象 URL 清理、网络失败、无文档状态和格式菜单。
- Playwright 覆盖立即导出未保存内容、DOCX/PDF 下载、手机布局和代码块菜单闪黑回归。

## 复制纯文本

- 从笔记复制到外部程序时，`text/plain` 直接使用编辑器可见文本，不再输出 Markdown 转义符或颜色 `<span>`。
- 保留段落、硬换行和代码块的换行；链接、行内代码、字体颜色与背景颜色只复制其可见文字。
- 测试覆盖命令行、下划线、等号、引号、颜色文本、标题、代码块、链接、图片替代文字和空选区。
