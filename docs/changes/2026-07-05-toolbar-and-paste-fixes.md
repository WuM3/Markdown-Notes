# 2026-07-05 工具栏与粘贴修复

## 修复内容

- 引用按钮改为可切换：光标或选区在引用内时再次点击会取消引用，不再无限嵌套引用块。
- 代码块按钮改为独立转换逻辑：选中文字后点击只把选中的内容转成代码块，空选区时仍转换当前块。
- “正文/标题”格式菜单移除了“代码块”，避免同一功能出现在两处。
- 选中文字工具栏新增“颜色”入口，可设置字体颜色、背景颜色，也可恢复默认。
- 颜色使用 Markdown 兼容的 HTML span 保存，重新打开文档时会解析回可编辑的颜色标记。
- 从 ChatGPT 等富文本来源复制“正文 + 多个代码块 + 正文”时，会保留全部段落和多个代码块，不再只留下第一个代码块。
- 普通多行文本粘贴继续保留换行，不会自动变成代码块。

## 验证

- `npm test`
- `npm run typecheck`
- `npm run lint`
