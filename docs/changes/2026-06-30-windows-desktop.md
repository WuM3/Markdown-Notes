# 2026-06-30 Windows 桌面版

## 新增功能

- 新增 Electron 桌面入口，Windows 程序可以自动启动本机笔记服务并打开独立窗口。
- 服务仍监听 `0.0.0.0:3210`，桌面版使用 `http://127.0.0.1:3210` 打开，其他局域网设备继续使用电脑 IPv4 地址访问。
- 首次启动桌面版时选择笔记数据目录，配置保存到 Electron `userData` 目录。
- 桌面版关闭窗口后最小化到系统托盘，托盘菜单支持打开窗口、复制局域网地址、更换数据目录、开机自启和退出。
- 如果 `3210` 已经有兼容的个人笔记服务，桌面版会直接连接；如果端口被其他服务占用，会提示错误。
- 优化窗口关闭行为：点击关闭按钮会销毁窗口并释放网页编辑器内存，托盘和本机服务继续保留；从托盘重新打开时再创建新窗口。
