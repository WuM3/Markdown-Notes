# 2026-07-13 响应式文档目录

- 桌面端文档目录宽度改为相对编辑区的弹性范围，不再根据浏览器视口 `vw` 固定计算，改善不同显示器、窗口和缩放下的布局稳定性。
- 所有端的长标题索引保持单行并显示省略号，避免目录与页面横向溢出。
- 移动端和触摸平板仅显示省略号，不显示悬停提示。
- 标题提示改为复用现有深色圆角全局提示样式，并且仅在桌面端标题实际被省略时显示；窗口、目录宽度或缩放变化后会重新判断是否需要提示。
- 修复触摸屏电脑接鼠标时被误判为触摸设备的问题：现在以浏览器的细指针悬停能力为准，标题实际溢出即可显示完整标题提示。
- 修复长标题提示文字超出黑色气泡的问题：提示可在最大宽度内换行，连续英文、URL 等无空格文本也会断行，不会横向溢出。
